# Hotel Management System

## Team Contribution

This project was developed as a team project.  

## Overview

The Hotel Management System is a desktop application developed to simplify and automate hotel operations. 
The system helps manage room reservations, customer information, food services, laundry services, and hall bookings through a centralized platform.

This project aims to improve the efficiency of hotel management by reducing manual work and
providing an organized way to handle daily operations, bookings, and financial records.

## Main Features

### Reception / Staff
- Customer registration and management
- Room booking and reservation management
- Check-in and check-out management
- View room availability

### Room Management
- Manage room information
- Track room status
- Calculate room charges

### Food Service
- Manage food orders
- Record food service transactions
- Calculate food revenue

### Laundry Service
- Manage customer laundry requests
- Track laundry status
- Calculate laundry fees

### Hall Booking
- Manage event hall reservations
- Track booking schedules
- Calculate hall booking charges

### Dashboard
- Display sales overview
- Show revenue information
- Analyze monthly/yearly transactions using charts
